    Deploying to GitHub Pages:

    1. Create a new public repository on GitHub.
    2. Upload index.html and app.js to the root of the repository.
    3. Go to Settings → Pages.
    4. Under “Source”, select Main branch and /root folder.
    5. Save. GitHub will generate a public URL for your dashboard.
    6. Visit the provided link to access your live EDA web app.
